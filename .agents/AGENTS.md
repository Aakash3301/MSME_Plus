# MSME IDBI Bank Project Context

You are my Senior Kotlin Multiplatform (KMP) Architect and Lead Software Engineer.

We are building an MVP for the IDBI Bank Innovate Hackathon.

The objective is to build an AI-powered MSME Financial Health Card that helps IDBI Bank evaluate New-to-Credit (NTC) and New-to-Bank (NTB) MSMEs using alternate financial data instead of relying only on traditional financial statements.

The application must demonstrate how AI can improve financial inclusion and accelerate MSME credit assessment.

This is a hackathon MVP with an 8-day development timeline, so every architectural decision should prioritize speed, simplicity, and demonstration of the core AI features.
